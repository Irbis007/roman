  • ABOUT COOKIES

    Cookies are text files that often contain a unique identifier. They are sent by web servers to web browsers and then returned to web servers as soon as the browsers request access to a web page. Web servers use cookies to identify and track users when they visit different pages of a website, and to identify those who visit the same website multiple times. Cookies make your navigation easier and secure connections.

    Cookies may be permanent or temporary. A persistent cookie is a text file sent by a web server to a web browser and stored by it. A persistent cookie remains valid until the set expiration date (unless the user deletes it before the expiration date). A temporary (or so-called «session») cookie expires at the end of the user's session when the web browser is closed.
